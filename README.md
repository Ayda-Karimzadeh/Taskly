# 🎀 Taskly

A simple command-line task manager built in Python. Add, view, and delete tasks with persistent storage in a local text file.

## Features
- Add one or multiple tasks at once (separated by `;`)
- View all current tasks
- Delete a specific task by number
- Clear all tasks
- Tasks are automatically saved to `tasks.txt` and reloaded on the next run

## How to run
```bash
python taskly.py
```
